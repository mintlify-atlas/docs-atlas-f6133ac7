# Welcome to Rihsac API

Rihsac API is a Laravel-based financial services platform that provides comprehensive solutions for loan applications, cash transactions (Cashir), and user profile management with integrated wallet functionality.

## Quick Start

Get started with Rihsac API in minutes:

<Steps>
  <Step title="Register a User">
    Create a new user account with email and password
    ```bash
    POST /api/register
    ```
  </Step>
  <Step title="Authenticate">
    Login to receive your JWT access token
    ```bash
    POST /api/login
    ```
  </Step>
  <Step title="Complete Onboarding">
    Verify email, phone, set transaction PIN, and complete profile
  </Step>
  <Step title="Make API Calls">
    Use your JWT token to access protected endpoints
  </Step>
</Steps>

## Technology Stack

Rihsac API is built with modern, battle-tested technologies:

- **Framework**: Laravel 8.x (PHP 7.3+/8.0+)
- **Authentication**: tymon/jwt-auth for JWT token management
- **Wallet System**: bavix/laravel-wallet for transaction handling
- **SMS Verification**: Twilio SDK integration
- **Image Processing**: Intervention/Image for document uploads

## Key Capabilities

### Authentication & Security
- JWT-based authentication with configurable token expiration
- Email and phone number verification
- Transaction PIN for sensitive operations
- Activity logging for user actions

### User Management
- Multi-stage onboarding process
- Profile completion tracking with `onboarding_stage` field
- Role-based access control (USER, ADMIN, etc.)
- Document upload and verification for KYC

### Financial Services
- Loan application management
- Cashir transaction processing
- Integrated wallet system with balance tracking
- Multiple payment channel support

### Profile Information
- **Guarantor Information**: Store and manage guarantor details
- **Employment Details**: Track employment history and status
- **Residential Information**: Maintain current address details
- **Banking Information**: Securely store bank account details

## API Endpoints Overview

The Rihsac API provides RESTful endpoints organized into logical groups:

```bash
# Authentication
POST   /api/login
POST   /api/register

# Onboarding (JWT Protected)
POST   /api/email/verify
POST   /api/phone/verify
POST   /api/transaction-pin
POST   /api/username/update

# Profile Management (JWT Protected)
POST   /api/profile/guarantor
GET    /api/profile/guarantor
POST   /api/profile/employment
GET    /api/profile/employment
POST   /api/profile/residential
GET    /api/profile/residential
POST   /api/profile/banking
GET    /api/profile/banking
```

<Note>
  All endpoints except `/login` and `/register` require JWT authentication via the `jwt.user` middleware.
</Note>
