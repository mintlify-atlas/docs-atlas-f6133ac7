---
title: 'Guarantor Information'
api: 'POST https://api.rihsac.com/api/profile/guarantor'
description: 'Store or update guarantor information for the authenticated user'
---

## Authentication

This endpoint requires JWT authentication. Include the bearer token in the Authorization header.

```bash
Authorization: Bearer {token}
```

## Request Body

<ParamField body="firstname" type="string" required>
  Guarantor's first name (max 100 characters)
</ParamField>

<ParamField body="lastname" type="string" required>
  Guarantor's last name (max 100 characters)
</ParamField>

<ParamField body="phone_number" type="string" required>
  Guarantor's phone number
</ParamField>

<ParamField body="email" type="string" required>
  Guarantor's email address (must be valid email format)
</ParamField>

<ParamField body="setting_id" type="integer" required>
  Relationship type ID (must exist in settings table)
</ParamField>

## Response

<ResponseField name="status" type="string">
  Response status: "success" or "failure"
</ResponseField>

<ResponseField name="message" type="string">
  Response message describing the result
</ResponseField>

<ResponseField name="data" type="object">
  Empty on successful store operation
</ResponseField>

<Expandable title="Success Response Example">

```json
{
  "status": "success",
  "message": "Guarantor information update successful",
  "data": {}
}
```

</Expandable>

<Expandable title="Error Response Example">

```json
{
  "status": "failure",
  "message": "Validation errors",
  "data": {
    "firstname": ["The firstname field is required."],
    "email": ["The email must be a valid email address."]
  }
}
```

</Expandable>

## Code Example

```bash cURL
curl -X POST https://api.rihsac.com/api/profile/guarantor \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "firstname": "John",
    "lastname": "Doe",
    "phone_number": "+2348012345678",
    "email": "john.doe@example.com",
    "setting_id": 1
  }'
```

# Get Guarantor Information

<api method="GET" endpoint="https://api.rihsac.com/api/profile/guarantor" />

Retrieve the guarantor information for the authenticated user.

## Authentication

This endpoint requires JWT authentication. Include the bearer token in the Authorization header.

```bash
Authorization: Bearer {token}
```

  Response status: "success" or "failure"
</ResponseField>

<ResponseField name="message" type="string">
  Response message describing the result
</ResponseField>
